Overview

This module enables the features of the various modules including Website Call For Price, Customer Order Comment, Ecommerce Barcode Search, Instagram Feed Snippet, Portal Dashboard, Website Clear Cart, Website Custom Contact Us, Hide Variants, Product Attachments on the Website, Website Return Order Management and Whatsapp Floating Icon in Website.

Features

Able to hide the product prices in Website Shop.
Users will be able to leave ratings for the products.
Products can be searched by scanning the barcode.
Instagram feed snippet.
Dashboard in Customer Portal.
Custom Contact Us form.
Able to hide product variants in Website Shop.
Product attachments in Website Shop.
Return of Sale Orders.
Whatsapp floating icon in Website.

Screenshots

1. Product Attachments on Website

Attach any product specific documents.

Visitors of your e-commerce portal can download these attachments.

2. Ecommerce Barcode Search

Go to Products -> PRINT LABELS. Give Barcode for the product and print the barcode using the PRINT LABELS option.

Go to Website Shop and clicking the BARCODE button will open the camera and scan the barcode.

Product of the scanned barcode will be displayed.

Website Call For Price

Go to Products -> Sales -> Enable Call for Price. After enabling this, the price and add to cart button will hide from the website.

Click on Call for Price button.

User need to enter these details inorder to get the price of the product. The request will send to merchant by clicking send button.

After sending the request, merchant can access the request from backend.

Initially, a draft request is created; after merchant verification, the sales team contacts the customer.

Website Clear Cart

Remove the items from the cart using the clear cart button in website.

Empty cart

Website Custom Contact Us

Go to Website -> Configuration -> Websites -> Contact Us

Set the visibility of company details based on activated the boolean fields on the website.

Go to Websites -> Home Page -> Contact Us Here you can see the customized contact us form

Portal Dashboard

Enable these settings and number of records to display in Portal.

Records are displayed in Portal.

Analysis pie chart for sales, purchase and accounting.

Hide Variants

Enable the checkbox to hide the product variant from the website.

The hidden product variant will be displayed in the tree view of the product variants.

The product is hidden and cannot be selected in the website

Instagram Feed Snippet

Go to Website > Configuration > Instagram.You can set up your instagram profile by adding access key and click fetch account button.Once the profile is generated click the fetch post button to fetch instagram post into odoo.

Goto the Instagram Posts menu in Website. All the Instagram Posts will be displayed there.

Visit the Odoo website frontend and click on the Edit option.After that, select the Insta Feed widget and drag it to the place where you want to showcase it on your Odoo website frontend, and click on the Save button.

Once you save it, the Instagram widget will display your Instagram feed on your Odoo website.

Customer Order Comment

You can enable customer order comment feature. Navigate to Website/ Configuration/ Settings/ Customer Order Comment and enable the feature.

Checkout any product from cart and customer can add rating and review in the order confirmation page.

The comment and rating will be created in "Customer Order Comment" tab in the Form View of the corresponding Sale Order.

Website Return Order Management

Goto Portal/ My Account -> Sale order , from the sale order click Return button.

You can select the product, quantity and return reason, after that you can submit the return order.

Customer can see list of RMA orders in Portal / My Account.

The customer can see the detailed information of the return order in portal.

Goto Sales -> Return Orders, where the Return Orders are created in the backend.

Form view of return order.

Shows all the delivery orders related with Sale order.

creation of Return picking from return order using confirm button.

Smart button for showing all return orders related to the sale order.

Print RMA PDF Report - Print Return Report.

Whatsapp Floating Icon in Website

In the website form view we can define the number along with country code required for communicating with the website responsible.

If the salesperson's signature is missing from the other info tab when establishing a new sale order, order confirmation will not be possible.

There will be a whatsapp icon in the website.
