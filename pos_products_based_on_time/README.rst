POS HAPPY HOURS | POS TIME BASED PRODUCTS
=========================================
This module enables you to display time-based products in POS. The item will
vary as time goes on.
 
Configuration
=============
The user should be added to the security group: Administrator(Point of sale)
in order to get access to the new menu.
